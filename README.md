# AVT-VQDB-UHD-2-HDR
An 8K HDR dataset published by the AVT group of TU Ilmenau

Download the sources: [https://avtshare01.rz.tu-ilmenau.de/avt-vqdb-uhd-2-hdr](https://avtshare01.rz.tu-ilmenau.de/avt-vqdb-uhd-2-hdr)
